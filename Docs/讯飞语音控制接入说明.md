# 讯飞语音控制接入说明

## 使用方式

运行 `MainScene` 后按住 `Y` 说话，松开 `Y` 后发送本次录音。识别结果和执行结果会显示在屏幕顶部，并写入 Unity Console。

当前支持：

- `油门百分之六十`、`油门六成`、`油门怠速`、`最大油门`
- `放下起落架`、`收起起落架`
- `襟翼增加一级`、`襟翼收回一级`、`襟翼全收`、`襟翼全放`
- `扰流板增加一级`、`扰流板收回一级`、`扰流板全收`、`扰流板全开`
- `开启刹车`、`解除刹车`
- `抬头配平`、`低头配平`
- `暂停飞行`、`继续飞行`

语音反推暂时禁用。起落架收起和刹车操作仍受 `FlightInput` 的本地安全条件限制。

## 配置凭据

凭据只从用户环境变量读取，不要写入 C#、Prefab、Scene 或其他会提交到 Git 的文件。

在 PowerShell 中执行以下命令，把占位内容替换为讯飞控制台重新生成的凭据：

```powershell
[Environment]::SetEnvironmentVariable("XFYUN_APP_ID", "你的APPID", "User")
[Environment]::SetEnvironmentVariable("XFYUN_API_KEY", "你的APIKey", "User")
[Environment]::SetEnvironmentVariable("XFYUN_API_SECRET", "你的APISecret", "User")
```

设置后完全退出并重新启动 Unity Hub 和 Unity Editor，让新进程读取环境变量。

## 讯飞控制台

应用需要开通“语音听写（流式版）WebAPI”。当前客户端连接地址为：

```text
wss://iat-api.xfyun.cn/v2/iat
```

音频使用 16000Hz、单声道、16bit、小端序 PCM，每 40ms 发送一帧。

## 发布注意事项

环境变量只能避免凭据进入仓库，不能防止最终用户从桌面程序中提取凭据。正式对外发布时，应把讯飞签名和请求代理放在自己的服务端，并为客户端增加身份认证、频率限制和用量监控。
